    ФОРМЫ

    <form name="имя" action="URL"(- задает адрес отправки формы) method="post / get"> ... </form> - объявление форм

    • get - данные из формы добавляются к URI атрибута action, их разделяет '?', и полученный URI посылается на сервер
    • post - посылает данные в теле HTTP-запроса и используется, когда нужно отправить много данных и ссылка на результат обработки этих данных не нужна. Например, при редактировании личного профиля

    <input type="тип поля" name="имя поля" id="уникальное имя на всей странице" /> - поля формы;

    • value="" - значение по умолчанию
    • placeholder="" - подсказка к полю формы
    • type="checkbox" checked(- вкл по умолчанию) - либо да либо нет
    • type="file" enctype="multipart/form-data" - указывает браузеру отправить на сервер данные по частям
    • type="hidden" - отправляет на сервер данные которые вводятся не пользователем
    • type="text" list="browsers" name="browser" - определить возможные значения
      <datalist id="browsers">
      ...
      <option value="firefox"> </option>
      <option value="vivaldi"> </option>
      ...
      </datalist">
    • pattern="[возможное значение]{кол-во этих значение}-[0-9]{3}--[a-b]{1}" - авто проверка формата
    • readonly - запись можно только читать и копировать(данные отправляются на сервер)
    • disabled - запись можно только читать, но не кипровать и редактировать (данные не отправляются на сервер)
    • autocomplete="on / off" - автозаполнение на основе ранее введенных данных
    • tabindex="1 / 2 / ..." - порядок навигации через tab
    • autofocus - устанавливает курсор по умолчанию
    • required - указать что поле обязательное
