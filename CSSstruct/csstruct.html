СТРУКТУРА CSS

СИНТАКСИС

! Вместо <link rel="stylesheet" href="style.css" /> можно прописать это в .html: 

                        
<head>
    <style>
        CSS-код
    </style>
</head>

Либо задавать стиль в атрибуте style - <div> style="width: 50%;"></div>

! Если в дочернем элементе прописать значение = 2em, то его значение будет в 2 раза больше основго и тд

Порядок важности свойств правил стилей: - браузерные -> теги -> классы -> ниже в коде 

 
                        
тег{
    правило: значение ;
} 
 
                        
.class{
    правило: значение ;
} 
                        
#id{
    правило: значение ;
}
 
                        
тег, .class{
    правило: значение ;
}
 
                        
nav a{
    правило: значение ; - выберет теги "a" внутри тегов "nav"
}
 
                        
input[type="text"]{
    правило: значение ; - выбор поля формы
}
 
html:

<div class="notif"> ... </div >
<div class="notif notif-bad"> ... </div >
.notif{
    ...
    color: red ; 
}
.notif-bad{
    color: blue ; - будут точно такие же правила как у .notif НО color будет другой 
}

СЕЛЕКТОРЫ

ВЫБОР ТЕГОВ С КЛАССОМ

HTML

                        
<ul class="target"></ul>

CSS

                        
ul.target {...} /* выбор всех тегов ul с классом target */

ВЫБОР ЭЛЕМЕНТОВ С ДВУМЯ КЛАССАМИ

HTML

                        
<span class="text green"></span>
<p class="green text"></p>

CSS

                        
.text.green {...} /* выбор элементов с двумя классами: text и green */

ВЫБОР ТЕГОВ С НЕСКОЛЬКИМИ КЛАССОМ

HTML

                        
<span class="underlined red big text"></span>

CSS

                        
span.underlined.red.big.text {...} /* выбор тегов span с четырьмя классами: underlined, red, big и text */

СОСЕДНИЕ СЕЛЕКТОРЫ

HTML

                        
<ul>
    <li class="hit"></li>
    <li class="miss"></li>
</ul>

CSS

                     
.hit + .miss / .hit + li / li + .miss / li + li  - применит стили ко всем элементам с классом miss после li / hit !НО .miss + .hit не сработает, тк элемент с классом miss находится после элемента с классом hit

ДОЧЕРНИЕ СЕЛЕКТОРЫ

HTML

                        
<ul>
    <li><span>...</span></li>
    <li><span>...</span></li>
</ul>

По отношению к списку <ul>: <li> - дочерние элементы и потомками, а <span> — потомки, !НО не дочерние элементы.

Чтобы обратиться только к дочерним элементам:

CSS

                        
ul > li{...} / ul > li > span{...}

ПСЕВДОКЛАССЫ

:first-child - первый дочерний элемент родителя

:last-child - последний дочерний элемент родителя 

                        
li:first-child{...} - выберет последний элемент списка

:nth-child( 1(- выберет первый) / 2(- второй) / 2n({/ even}- все четные) / odd(- нечетные) / 2n+1(n: 0) 2*0+1 | 1) 2*1+1 | ... )) / nth-child(n+3):nth-child(-n+4) (- 3 и 4 элемент) - выберет заданный в скобках элемент; 

                        
li:nth-child(2n) { ... }
.shooter-2 li:nth-child(3) { ... }

:hover - применяет стили после наведения мыши | селектор1:hover селектор2 

                        
h3:hover{
    background-color: #fcf8e3;
}

Динамические эффекты

                        
li.top ul.submenu {
    display: none;
}
                              
li.top:hover ul.submenu {
    display: block;
}

:link - выбирает еще не посещенные ссылки

:visited - посещенные ссылки

:active - активные ссылки

                            
a:link {...}
a:visited {...}
a:hover { ... } - желательно в таком порядке
a:active {...}

:focus - выбирает элемент, который в фокусе / куда установлен крусор / выбранный через tab элемент

                            
input:focus {...}

СЕЛЕКТОРЫ АТРИБУТОВ

input[checked] { ... } - выберет все поля формы где есть атрибут checked

input[type="text"] {...} - поля формы у которой атрибут type="text"

input[value="Войти"] - поля формы со словами Войти

ПРАВИЛА

ОФОРМЛЕНИЕ

color: #123456; - основной цвет

background-color: #0000; - цвет заднего фона

font: font-size line-height font-family font-weight font-style font-variant; - задавание текста

  • font-size: 10px; - размер текста
  • line-height: 26px / 120% / 2 ( - font-size*2 ) / normal( - вычисляется браузером); - высота строки
  • font-family: "Arial", sans-serif ( - второй, менее нужный шрифт); - шрифт
  • font-weight: normal / bold ( - жирный); - насыщенность/ширина шрифта
  • font-style: normal / italic(- курсви) / oblique(- наклонное); начертание
  • font-variant: normal;

white-space: значение ; - управление пробелами

  • nowrap - схлопывает лишние пробелы и отображает весь текст одной строкой без переносов
  • pre - как <pre>
  • pre-wrap - как <pre> НО если текст длинне ширины страницы - добавит перенос
  • normal - режим по умолчанию: лишние пробелы и переносы строк схлопываются, текст переносится, пробелы в конце строк удаляются

text-transform: none / lowercase(- все строчные) / uppercase(- все заглавные) / capitalize(- каждое слово начинается с заглавной ) ; - управление регистром

text-decoration: LINE: значение1 значение2 COLOR:#fff STYLE: типЛинии( solid / double / dotted / dashed(- пунктирная) / wavy(волнистая)); - подчеркивание

  • none - убирает все
  • underline - подчеркивание, как <u>
  • line-through - зачеркивание, как <del>
  • overline - надчеркивание

text-align: left / right / center / justify(- по всей шрине блока); - горизонтальное выравнивание внутри блока

vertical-align: 100% / 10px / top / middle / bottom(= 0% / 0px)(- по нижнему краю) / baseline(- по базовой линии); - горизонтальное выравнивание по строке

margin: СВЕРЗУpx СПРАВА% СНИЗУpx СЛЕВА%; - внешние оступы

padding: СВЕРЗУpx СПРАВА% СНИЗУpx СЛЕВА%; - внутренние отступы
